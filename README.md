# Credit Scoring Model

## Project Description
This project predicts whether a customer is likely to be a credit risk using machine learning.

## Dataset
German Credit Risk Dataset (Kaggle)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm
Random Forest Classifier

## How to Run
1. Install requirements
2. Run:
   python credit_score_model.py

## Results
- Accuracy Score
- Classification Report
- Confusion Matrix
- Feature Importance Graph
